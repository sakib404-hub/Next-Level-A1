# Advanced Problem Solving with TypeScript & OOP

## 📌 Overview
This repository contains solutions to a set of TypeScript problems designed to demonstrate core programming concepts such as type safety, generics, interfaces, and object-oriented programming (OOP).

## 💻 Implemented Problems

- Filter even numbers from an array  
- Reverse a string  
- Type checking using union types and type guards  
- Generic function for object property access  
- Interface-based object transformation  
- Class inheritance (Person → Student)  
- Array intersection logic
## ⚙️ Technologies Used

- TypeScript  
- OOP (Inheritance, Encapsulation)
