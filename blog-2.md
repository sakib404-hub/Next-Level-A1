                                                 OMIT & PICK
                                                
Pick and Omit are utility types that used to create new types from an existing interface without rewriting code. 
Pick selects only specific properties from a type.
Omit removes specific properties from a type.
These utility types help developers reuse existing interfaces efficiently, reduce code duplication, and maintain cleaner and more manageable code. By using Pick and Omit, TypeScript supports the DRY (Don’t Repeat Yourself) principle, making applications easier to update and maintain.
